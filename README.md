# 🍽️ Restaurant Premium

Página web completa para un restaurante, con menú de comidas, bebidas, sistema de reservas y un juego interactivo integrado.

## 📌 Descripción del problema

Los restaurantes necesitan una presencia digital atractiva que permita a sus clientes consultar el menú, conocer las bebidas disponibles, hacer una reserva en línea y tener una experiencia interactiva agradable, todo desde un mismo sitio web.

## 💡 Solución

Se desarrolló un sitio web con navegación completa entre secciones:
- **Inicio:** presentación del restaurante
- **Comidas:** platos del menú con ingredientes y fotos (Filete Mignon, Venado con membrillo, entre otros)
- **Bebidas:** carta de bebidas disponibles
- **Reservas:** formulario para reservar una mesa
- **Juego:** minijuego interactivo integrado en la página

## 🛠️ Tecnologías

- HTML5
- CSS3
- JavaScript (vanilla)
- DOM API
- Formularios HTML

## ⚙️ Instalación y uso

1. Clona el repositorio
2. Abre la carpeta del proyecto
3. Abre el archivo `index.html` en tu navegador
4. No requiere instalación de dependencias ni servidor
